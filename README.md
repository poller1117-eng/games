## 製作說明

### 一、選用的美術設計

依企劃書「黑暗水彩幻想」風格，本日採用以下美術方案：

1. **角色繪製：向量化即時 SVG 幽靈造型**
   - 阿幽以 `<canvas>` 即時繪製（雙頂髮絲、晃動衣襬、靈體下襬尾巴）
   - 周身放射綠青色 radial gradient 光暈
   - 內建 5 款皮膚色版（經典靈衣／人魚／雪精靈／賽博朋克／櫻緋）

2. **場景配色（依企劃 §3.2）**
   - 主色 `#1A0D3A`、輔色 `#4A7C8E`、強調色 `#00FFCC`、危險 `#C0392B`
   - 五個區域各自獨立調色：入口（紫）、水鏡（青）、焰獄（橙紅）、暗林（暗紫）、風塔（綠）

3. **視差背景**
   - 三層 silhouette 山型遠景，依攝影機移動產生視差
   - 環境粒子（30 顆）漂浮模擬靈氣

4. **UI 美術**
   - HUD 使用半透明黑底＋幽靈青描邊，符合「極簡浮層」原則
   - 商城／地圖介面採古卷軸風格（米黃 `#e8d9b0` 紙紋＋朱褐 `#3a1f0a` 邊框）
   - 字體：思源黑體 + ZCOOL QingKe HuangYou 做為標題毛筆體

5. **平台與敵人**
   - 石材平台兩段漸層（青灰系），熔岩平台改用橘紅漸層+熱效標記
   - 敵人為發光「靈體」造型，依屬性著色，下方有抖動尾翼
   - Boss「羽翼守衛」雙翅振動＋紅眼，並繪製血條與名字

### 二、實作的核心玩法（依企劃 §2.2）

- ✅ **靈能汲取（Soul Absorption）** — 擊敗敵人吸收靈魂碎片，自動裝備兩種屬性
- ✅ **能力解鎖探索（Ability-Gated）** — 水/火/暗能力開啟封印門
- ✅ **Boss 靈核** — 擊敗 Boss 觸發勝利畫面（試玩版簡化）
- ✅ **大地圖系統** — Tab 開啟古卷軸式房間圖，已探索墨色填充、未探索留白
- ✅ **靈能組合傷害加成** — 屬性匹配敵人額外傷害；雙屬性裝備再 +1
- ⚪ 靈界留言板、賽季速通：因 H5 單機版限制，UI 預留但後端未串接

### 三、商城設計（依企劃 §5.2，三類「不影響平衡」付費道具）

**所有道具皆可用遊戲內「靈幣」購買，或對應原價新台幣參考。**

| 分類 | 道具 | 對應企劃條目 |
|---|---|---|
| 皮膚外觀 | 人魚幽靈、雪精靈、賽博朋克、櫻緋幻影 | §5.2 阿幽皮膚包 NT$60-120 |
| 便利道具 | 復活之泉、地圖標記擴充包、靈珠雙倍卡、古卷地圖皮膚 | §5.2 復活之泉 NT$9／標記包 NT$39 |
| 通行證   | 靈界密探月票、幽靈祭典賽季通行證、神殿典藏版 | §5.2 月票 NT$99／典藏 NT$199 |

商城特色：
- **不影響平衡**：皮膚純外觀；復活之泉為便利性、不增加數值
- **雙幣制**：可用真實貨幣或遊戲內貨幣購買，符合「溫和付費」哲學
- **可預覽**：點擊右上「靈幣」即開啟商城

### 四、操控（依企劃 §2.4）

**桌機（鍵盤）：**
- 移動：方向鍵 / `WASD`
- 跳躍：`Space` / `Z`（解鎖「暗」能力後可二段跳）
- 攻擊：`X`
- 技能：`C`（消耗 25 SP，依首選屬性釋放不同魔法）
- 切換屬性：`Q` / `E`
- 開啟地圖：`Tab`
- 開啟商城：`B`

**手機（觸控）：**
- 左下虛擬搖桿移動
- 右下按鈕：跳、攻擊、技、切換屬性
- 左中「地圖」按鈕

### 五、遊戲流程（試玩版）

1. 入口迴廊（無能力區）— 擊敗 2 隻基本靈體取得首屬性
2. 水鏡祭廳 → 擊敗水靈獲得「水」能力，可通過熔岩
3. 焰獄鍛造間 → 擊敗火靈獲得「火」能力
4. 暗林靈苑 → 擊敗暗靈獲得「暗」能力，解鎖二段跳
5. 風塔迴廊 → 挑戰 Boss「羽翼守衛」，獲勝即通關試玩版

完整版含 7 大區域＋6 名 Boss＋最終 Boss 「神殿意志」，於本企劃 §2.3 中規劃。
